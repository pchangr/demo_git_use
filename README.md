# 🚀 [Anontrace]

> [plataforma web de visualización de datos ciudadanos que centraliza reportes de seguridad para permitir a los vecinos consultar zonas de riesgo y registrar incidentes de manera colaborativa"]

---

## 🌟 Características

- ✨ **Interfaz Intuitiva:** Diseñada para ser fácil de usar desde el primer segundo.
- 📱 **Responsive:** Se ve increíble en móviles, tablets y escritorio.

## 🛠️ Tecnologías Utilizadas


| Categoría | Tecnología |
| :--- | :--- |
| 🎨 Frontend| JavaScript(ES6+), CSS3, HTML5| 
| ☕ Backend| Java (Spring Boot / Servlets)| 
| 🗄️ Database| T-SQL (Microsoft SQL Server)| 
| 📍 Mapas| Leaflet.js, OpenStreetMap| 
| 🔔 UI/UX| SweetAlert 2, FontAwesome| 
| ⚙️ Herramientas| Git, Maven/Gradle, JDBC Driver| 

## 🚀 Instalación y Uso

¡Ponlo en marcha en menos de 2 minutos!

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/pchangr/demo_git_use
   ```

2. **Descarga y ejecuta la base de datos:**
   ```bash
   Script de bd: https://comunidadupnedu-my.sharepoint.com/:u:/g/personal/n00530358_upn_pe/IQDHhxuwLgsCSYPQiR5UemiAAQnWG8JxeP2z0Yx4UZ9-Gxc?e=Mz8zuJ
   Modifica el script de conexion en el proyecto: com.anontrace.utils.configDB 
   ```

3. **Ejecuta el servidor:**
   ```bash
   Ejecuta el servidor de Tomcat Apache 9.0.40
   ```

4. **¡Dale play!:**
   ```bash
   Ejecuta la web http://localhost:8082/anontrace-web/
   ```
